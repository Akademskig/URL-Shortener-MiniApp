Url shortener microservice

A small app that returns a shortened URL if an URL is passed as a parameter. When a shortened URL is visited it redirects the user to the original URL.

Example usage:

  • https://url-shortener-miniapp.herokuapp.com/url/https://www.google.com
  • https://url-shortener-miniapp.herokuapp.com/url/https://www.biserka.com

Example output:

  • { "original_url":"http://google.com", "short_url":"https://url-shortener-miniapp.herokuapp.com/1" }

Expected result:

Visiting https://url-shortener-miniapp.herokuapp.com/1 redirects to http://google.com.
